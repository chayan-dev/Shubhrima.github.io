Writerly

This is a website for a content-writing company. The company Writerly follows the motto, You Think, We Pen, You Smile. The website has features like sample articles, sign in page, payment page, and the about us section. There is a join us section, where freelance writers interested to be a part of the team can create their Writerly account. It also allows you to signup for newsletters.

Tech Stack:
  • HTML5
  • CSS3
  • Bootstrap
  • JavaScript
  • Deployed on Heroku
  • Subscribe newsletters on Mailchimp
  • Node.js
